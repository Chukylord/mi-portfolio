# Portfolio profesional

Landing personal desarrollada con **HTML + CSS** para presentar perfil profesional, stack técnico, proyectos destacados y vías de contacto.

## Secciones
- Hero con propuesta de valor y llamados a la acción.
- Perfil profesional y fortalezas.
- Stack técnico.
- Proyectos destacados.
- Contacto directo por email.

## Ejecutar en local
Al ser un sitio estático, podés abrir `index.html` directamente o levantar un servidor simple:

```bash
python3 -m http.server 4173
```

Luego visitá `http://localhost:4173`.
